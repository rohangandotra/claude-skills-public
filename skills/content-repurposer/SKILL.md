# Content Repurposer

## Overview
Takes one piece of long-form content and generates 10+ ready-to-post pieces across multiple platforms, each formatted for that platform's algorithm and audience expectations.

## When to Activate
- User provides a video transcript, blog post, or long-form text
- User asks to "repurpose", "turn into posts", "create social content from"
- User mentions wanting content for multiple platforms

## Process

### Step 1: Analyze the Source Content
- Identify the 3-5 key insights or talking points
- Find the most quotable/shareable moments
- Note any data points, stats, or bold claims
- Identify the emotional peak moments

### Step 2: Generate Platform-Specific Posts

For each platform, follow these rules:

#### Instagram Reels Caption
- Lead with a hook (pattern interrupt, curiosity gap, or bold claim)
- Keep under 2,200 characters but front-load the first 125 characters (preview cutoff)
- End with a CTA: "Comment [WORD] and I'll send you [resource]"
- Add 20-30 relevant hashtags in a separate paragraph
- Tone: casual, direct, use line breaks for readability

#### TikTok Caption
- Under 300 characters
- Hook in first line
- 3-5 hashtags max, mix niche + broad
- Conversational tone, can use slang

#### Twitter/X Thread
- First tweet is the hook (under 280 chars, must stand alone)
- 4-7 tweets in the thread
- Each tweet is one complete idea
- End with "Follow for more [topic]" or "RT if you agree"
- No hashtags in the thread body, 1-2 in the last tweet

#### LinkedIn Post
- Professional but not corporate
- Start with a bold statement or contrarian take
- Use single-sentence paragraphs for readability
- 1,300 characters max for full visibility without "see more"
- End with a question to drive comments

#### YouTube Shorts Description
- Under 100 characters for the title
- 2-3 sentence description with keywords
- Include relevant tags

### Step 3: Generate Hook Variations
For the top 3 posts, create 2 alternative hooks each so the user can A/B test.

### Step 4: Create a Posting Schedule
Suggest which post to publish on which day/time based on platform best practices:
- Instagram: 11am-1pm or 7-9pm
- TikTok: 7-9am, 12-3pm, or 7-9pm
- Twitter: 8-10am or 12-1pm
- LinkedIn: 7-8am or 5-6pm Tuesday-Thursday
